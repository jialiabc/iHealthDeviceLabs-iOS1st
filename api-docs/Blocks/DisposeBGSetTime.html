BG5 errorID: 00：Battery is low. 01：Glucose test result is out of the measurement range. 02：2.5V reference voltage error, not normal measurement, please repeat the test. 03：Strip is used or unknown moisture detected, discard the test strip and repeat the test with a new strip. 04：Reading transmission error.
